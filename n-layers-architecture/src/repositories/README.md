# Repositories

- Abstracciones del conjunto de entidades del dominio como colecciones.
- Ofrecen operaciones significativas en el lenguaje del dominio (identidad, criterios).
- Ocultan persistencia y topología; aíslan al dominio de tecnología.
- Sostienen políticas de consistencia y transacciones del modelo de negocio.
- Se expresan como interfaces del dominio; implementaciones viven fuera.
