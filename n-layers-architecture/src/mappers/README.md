# Mappers

- Puentes semánticos entre modelos (DTO ↔ Entidad ↔ Vista).
- Preservan el lenguaje del dominio al cruzar límites técnicos.
- Materializan Anti-Corruption Layer cuando se interactúa con modelos externos.
- Deciden proyección y normalización sin alterar reglas de negocio.
- Deterministas e idempotentes; no introducen efectos laterales.
