# DTOs

- Contratos de intercambio entre capas y límites del sistema.
- Modelo de transporte, no de dominio; sin comportamiento ni invariantes.
- Definen forma y significado público de datos y son versionables.
- Protegen al dominio de cambios de interfaz y preocupaciones externas.
- Expresan contextos y audiencias (API, eventos, persistencia).
- Buscan estabilidad y claridad semántica en el intercambio.
