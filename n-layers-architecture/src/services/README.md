# Services

- Servicios de aplicación: coordinan casos de uso y políticas de proceso.
- Orquestan colaboraciones entre entidades, repositorios y servicios del dominio.
- Definen límites transaccionales y criterios de éxito/falla del proceso.
- No contienen detalles de transporte ni persistencia; dependen de contratos.
- Sin estado y orientados al lenguaje ubicuo; retornan resultados del proceso.
